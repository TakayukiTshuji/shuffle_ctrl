# シャッフルコントロール

カードベースゼロ知識証明（Physical Zero-Knowledge Proof）のプロトコルを、物理的なカードを使わずにPC上でシミュレーション・実演するためのデスクトップアプリケーションです。

## 📖 概要 (Overview)
このアプリケーションは、**「カードベースゼロ知識証明のプロトコル」** の検証およびデモンストレーションを行うために開発されました。

物理的なカードを用いたゼロ知識証明は、直感的で理解しやすい反面、カードの準備や手作業でのシャッフルに手間がかかります。本アプリを使用することで、以下のメリットが得られます。

- **操作の可視化**: カードの配置や「裏返す」「シャッフルする」といった操作を画面上で再現。
- **手軽な検証**: 物理カード不要で、いつでもどこでもプロトコルの実演が可能。

## ✨ 主な機能 (Features)
- **カード操作**: ドラッグ＆ドロップによるカード配置、クリックによる裏返し操作。
- **ランダムシャッフル**: コンピュータ上でのランダム置換によるシャッフル機能。
- **正当性検証**: プロトコルに従った手順で、証明が受理されるかどうかの判定（実装済みの場合）。

## 🚀 インストールと使い方 (Download & Usage)

### 一般ユーザー向け (Windows)
このリポジトリの **[Releases](https://github.com/TakayukiTshuji/shuffle_ctrl/releases)** ページから、最新のインストーラー（`.exe`）をダウンロードしてください。

1. `cardshuffle-0.1.3-setup.exe` をダウンロード。
2. ファイルを実行するとインストールが完了し、アプリが起動します。

### 開発者向け (For Developers)
ソースコードから実行・ビルドを行う場合は以下の手順に従ってください。

```bash
# リポジトリのクローン
git clone https://github.com/TakayukiTshuji/shuffle_ctr.git
cd [リポジトリ名]

# 依存パッケージのインストール
npm install

# 開発モードで起動
npm start

# ビルド（Windows用インストーラー作成）
npm run make
```
